# Project: ShopEase Store Landing Page

## Design Summary
This landing page was built using Bootstrap CSS. It includes a sticky header, hero section with background image, and a responsive grid layout for four product cards. The layout uses mobile-first design and utility classes for clean, modern styling. All sections scale appropriately across devices, and buttons use hover effects for interactivity.

## Technologies Used
- HTML5
- Bootstrap CSS
- Google Fonts (Roboto)
